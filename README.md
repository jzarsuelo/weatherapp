# Weather App

Coding exam for Home Credit 

## Supported Version

- Target SDK Version - Android 7.1.1 (API Level 25)
- Minimum SDK Version - Android 4.4 (API Level 19)

## Prerequisite
- Android Studio 2.3.3 - You can install or update the items below via **Android SDK Manager**, under **SDK Tools**
  - Android Build Tools Version - 26.0.0
  - Android Support Repository - 47.0.0


## Installation

1. Clone the project repository

```sh
$ git clone https://github.com/jzarsuelo/weatherapp.git weatherapp
```

2. Import the project in Android Studio

3. Build and run the project. Make sure the Android version of the device or emulator is supported.
